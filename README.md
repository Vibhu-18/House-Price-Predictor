# 🏠 House Price Prediction Web App

This is a Machine Learning web application that predicts house prices based on property features.

## 🚀 Live Demo
[Click Here to Use the App](https://house-price-predictor-pfmkzqcy2pswqj5ggprint.streamlit.app/#house-price-prediction-app)

## 📌 Features
- Trained using Random Forest Regressor
- Dynamic input fields
- Real-time price prediction
- Deployed on Streamlit Cloud

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## 📊 Model Performance
R2 Score: 0.96
